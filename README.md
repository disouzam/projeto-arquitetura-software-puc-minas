# Propósito

Esse repositório contém algumas atividades executadas para a disciplina Projeto de Arquitetura de Software, do currículo do curso de [Arquitetura de Software Distribuído](https://vemprapuc.pucminas.br/arquitetura-de-software-distribuido-online), lecionada pelo professor [Marco Mendes](https://github.com/marco-mendes), oferta 9, turma 1, 2025/2026.

O conteúdo presente aqui é fruto da minha escrita individual, suportada por material educacional disponível na internet (com referências mencionadas sempre que necessário) e também faz uso de modelos de linguagem natural disponíveis (o prompt e o conteúdo na íntegra será disponibilizado em arquivos separados)
